# CloudTube - Project Information

## 📖 About

**CloudTube** is a Telegram bot that automates YouTube video downloads to WebDAV storage.

### 🎯 Mission

Make it easy for anyone to save their favorite YouTube content to their personal cloud storage.

## 🏗️ Architecture

- **Backend**: Python with asyncio
- **Bot Framework**: python-telegram-bot
- **Downloader**: yt-dlp
- **Storage**: WebDAV (webdav4)
- **Database**: SQLite (aiosqlite)
- **Monitoring**: psutil

## ✨ Key Features

1. **Download Management**
   - Single videos
   - Full playlists
   - Quality selection
   - Progress tracking

2. **Storage Integration**
   - WebDAV support
   - Automatic organization
   - Space checking
   - Reconnection handling

3. **User Experience**
   - Telegram interface
   - Customizable notifications
   - Task management (pause/resume/cancel)
   - History viewing

4. **Reliability**
   - Retry logic
   - Error handling
   - State recovery
   - Resource monitoring

5. **Cross-Platform**
   - Windows 10/11
   - Ubuntu 22.04/24.04
   - macOS
