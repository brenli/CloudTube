## 🚀 Как начать использовать

### Шаг 1: Установка (если еще не сделано)

```bash
# Автоматическая установка
chmod +x install.sh
./install.sh
```

### Шаг 2: Получение OAuth токена

```bash
# Запустите скрипт
python get_yandex_token.py
```

Или получите вручную: https://oauth.yandex.ru/authorize?response_type=token&client_id=23cabbbdc6cd418abb4b39c32c41195d

### Шаг 3: Настройка бота

```bash
# Отредактируйте .env
nano .env
```

Заполните:
```
TELEGRAM_BOT_TOKEN=ваш_токен
TELEGRAM_OWNER_ID=ваш_id
```

### Шаг 4: Запуск

```bash
source venv/bin/activate
python -m bot.main
```

### Шаг 5: Подключение Яндекс.Диска

В Telegram напишите боту:
```
/connect https://webdav.yandex.ru email@yandex.ru oauth_token
```

### Шаг 6: Первая загрузка

Отправьте ссылку на видео:
```
https://www.youtube.com/watch?v=dQw4w9WgXcQ
```

## 🎯 Основные возможности

### Автоматическое монтирование
Бот автоматически монтирует Яндекс.Диск при команде `/connect`:
- ✅ Сохраняет учетные данные в `~/.davfs2/secrets`
- ✅ Монтирует диск с правильными правами (uid/gid)
- ✅ Проверяет доступность и свободное место

### Организация файлов
```
Яндекс.Диск/
├── Single Videos/     # Отдельные видео
└── Playlist Name/     # Плейлисты
```

### Прогресс загрузки
- 0-90%: Скачивание с YouTube
- 90-100%: Загрузка на Яндекс.Диск

### Команды управления
```
/status              - Статус системы
/history             - История загрузок
/pause <task_id>     - Приостановить
/resume <task_id>    - Возобновить
/cancel <task_id>    - Отменить
```

## ⚠️ Важные замечания

### OAuth токен
- ⏰ Срок действия: 1 год
- 🔄 Обновление: `python get_yandex_token.py`
- 🔒 Доступ только к Яндекс.Диску

### Права доступа
- 📁 Точка монтирования: `/mnt/yandex-disk`
- 👤 Владелец: текущий пользователь
- 🔐 Файл учетных данных: `~/.davfs2/secrets` (права 600)

### Производительность
- 📊 Параллельные загрузки: 2 (по умолчанию)
- 💾 Чанки загрузки: 1MB
- 🔄 Автоматические повторы: 3 попытки

## 🐛 Устранение проблем

### Диск не монтируется
```bash
# Проверьте учетные данные
cat ~/.davfs2/secrets

# Проверьте права
ls -la ~/.davfs2/secrets  # должно быть -rw-------

# Попробуйте вручную
sudo mount -t davfs https://webdav.yandex.ru /mnt/yandex-disk -o uid=$UID,gid=$GID
```

### Ошибка загрузки
```bash
# Обновите yt-dlp
./update_ytdlp.sh

# Проверьте логи
tail -f logs/bot.log

# Проверьте место на диске
/status
```

### YouTube блокирует
```bash
# Обновите yt-dlp до последней версии
pip install --upgrade yt-dlp

# Перезапустите бота
sudo systemctl restart cloudtube
```
